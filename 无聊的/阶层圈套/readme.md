[为什么经历越多，越感觉这个社会比什么都恶心？ - 上善在若水的回答 - 知乎](https://www.zhihu.com/question/456031339/answer/1907164836816748689)

每个阶层看似在“向上爬”，其实都在被自己所在位置的规则、认知、资源限制和外部收割机制套牢。简单粗暴地分层说说常见的“坑”：

**底层（生存型）** 最常见的圈套是：

- “努力就能翻身”的工业化叙事 → 但实际是低端劳动力无限内卷，时间换钱的天花板极低
- 消费主义+高利贷+短视频成瘾 → 把本来就少的钱和注意力榨干
- “人坑人”环境 → 信息不对称、被中介/包工头/培训机构反复收割
- 最大的坑：**把所有问题都归结为“不够努力”**，从而拒绝系统性思考

**中产（焦虑型）** 这是目前被讨论最多、也最容易自我麻醉的阶层：

- “教育投资回报率”的信仰 → 砸几百万读名校/国际学校/各种班，最后发现岗位天花板比想象中低得多
- “阶层保卫战” → 买学区房、卷课外培训、卷简历、卷35岁危机、卷孩子985/211/出国
- 内卷式竞争 → 大家都卷，结果是集体向下流动的概率大幅提高
- 认知圈套：**把“比上不足比下有余”当成安全感**，却没意识到整个中产正在被“软着陆”到软阶层

**准精英 / 高净值（资源型）** 很多人以为到了这一层就安全了，其实坑更深、更隐蔽：

- “永远不够”的游戏 → 资产规模越大，保值增值压力越大，睡不好觉的不是穷人，反而是这一层
- 关系与站队风险 → 资源和机会高度依赖圈层，一旦站错队/风向变，掉得比谁都快
- “人抬人”的假象 → 表面是互相抬轿，实际是互相绑架，退出成本极高
- 最大的坑：**把“资源=自由”当成真理**，结果发现资源越多、被绑定的链条越多

**真正顶层（规则型）** 这个层级基本不公开讨论，但公开的信息已经足够说明问题：

- 最大的圈套其实是“必须持续做局”
- 一旦停止扩张或守成，就会被下一波规则制定者收割
- 家族传承的陷阱 → 二代三代能力跟不上资源，衰败速度比想象中快
